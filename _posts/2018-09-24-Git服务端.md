## Git学习之Git服务端

### Git安装
安装环境:
```bash 
[root@localhost ~]# uname -a
Linux localhost.localdomain 2.6.32-431.el6.x86_64 #1 SMP Fri Nov 22 03:15:09 UTC 2013 x86_64 x86_64 x86_64 GNU/Linux
```
安装git:
```bash
yum install git
```
